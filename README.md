# Frontend Tech Knowledge Base

前端技术知识库 - 包含完整的前端开发核心知识和技术文章

## 📚 项目介绍

这是一个基于 Vue 3 + TypeScript + Vite 构建的前端技术知识库，包含 19 份完整的前端开发核心知识，涵盖 Vue、JavaScript、TypeScript、Node.js、数据可视化、AI 等多个技术领域。

## ✨ 特性

- 📖 **19 份完整核心知识** - 涵盖前端开发各个领域
- 🎯 **系统化学习** - 按主题分类，循序渐进
- 💡 **实战导向** - 包含大量代码示例和最佳实践
- 🚀 **快速上手** - 基于现代技术栈，易于理解和扩展
- 📱 **响应式设计** - 支持多种设备访问
- 🔍 **易于搜索** - 完整的分类和索引

## 🚀 快速开始

### 环境要求
- Node.js 18.0+
- npm 9.0+ 或 pnpm 8.0+

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:5173 查看博客

### 构建生产版本
```bash
npm run build
```

## 🛠️ 开发指南

### 项目结构
```
frontend-blog/
├── posts/          # 核心知识文章
├── src/            # 源代码
├── about/          # 关于页面
├── assets/         # 静态资源
└── README.md       # 项目说明
```

### 技术栈
- **Vue 3.4+** - 渐进式 JavaScript 框架
- **TypeScript 5.3+** - JavaScript 超集
- **Vite 5.0+** - 新一代构建工具
- **Vue Router 4.2+** - 官方路由管理器
- **Pinia 2.1+** - 官方状态管理库

### 开发命令
```bash
# 开发
npm run dev

# 构建
npm run build

# 预览
npm run preview

# 类型检查
npm run type-check
```

详细开发文档请参考 [开发文档](./DEVELOPMENT.md)
