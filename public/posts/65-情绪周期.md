# 情绪周期

情绪周期是股票市场中投资者情绪从悲观到乐观、再从乐观到悲观的循环过程。理解情绪周期有助于把握市场节奏和个股机会。

## 情绪周期阶段

### 1. 冰点期（恐慌期）
- 市场普遍悲观，成交量萎缩
- 利好消息被忽视，利空被放大
- **策略**：分批建仓，关注超跌优质股

### 2. 复苏期（犹豫期）
- 市场开始企稳，热点出现
- 投资者信心逐步恢复
- **策略**：布局主线热点，持股待涨

### 3. 高潮期（狂热期）
- 市场全面上涨，赚钱效应明显
- 新手跑步入场，杠杆资金活跃
- **策略**：逐渐减仓，警惕见顶信号

### 4. 退潮期（悲观期）
- 上涨乏力，热点持续性差
- 投资者开始亏损
- **策略**：轻仓或空仓，等待下一次机会

## 情绪指标

### 量化指标
- 涨跌停家数对比
- 昨日涨停股今日表现
- 炸板率
- 跌停家数
- 成交量变化

### 情绪温度计
```
市场情绪温度 = (上涨家数 / 下跌家数) × 成交量因子 × 热点持续性
```

## 实战应用

### 连板股情绪
- 首板、二板家数变化
- 连板股晋级率
- 龙头股溢价情况

### 板块情绪
- 龙头股走势
- 跟风股表现
- 板块内个股联动性

### 周期切换信号
1. 空间板高度变化
2. 核心个股走势
3. 板块轮动节奏
4. 成交量配合情况

## 注意事项

- 情绪周期并非完全可预测
- 结合基本面综合判断
- 控制仓位，防范风险
- 尊重市场，顺势而为
