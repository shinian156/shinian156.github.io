## 龙头股特征

### 基本特征
1. **先于大盘启动**：板块或大盘启动时，龙头股率先上涨
2. **涨幅最大**：在同一板块中涨幅领先
3. **持续性强**：涨势持续，不轻易结束
4. **换手率适中**：资金接力良好
5. **带动性强**：能带动板块内其他股票上涨

### 识别龙头股
```
龙头股识别要素：
✓ 涨停时间早
✓ 封单量大
✓ 涨停质量高（不炸板）
✓ 带动跟风股
✓ 成交量的配合
```

## 龙头股分类

### 1. 市场总龙头
- 整个市场的最高连板股
- 代表市场最高空间
- 关注度最高

### 2. 板块龙头
- 特定板块的领涨股
- 带动整个板块炒作
- 需要结合热点判断

### 3. 卡位龙头
- 在主线调整时卡位
- 承接主线资金
- 周期较短

### 4. 补涨龙头
- 主线高位时启动
- 补涨属性
- 高度有限

## 龙头战法要点

### 1. 买入时机
- **首板**：题材发酵当日
- **二板**：确认龙头地位
- **分歧转一致**：高位震荡后再次涨停
- **龙回头**：调整后的第二波

### 2. 卖出时机
- **破5日线**：短线走弱
- **高位放量大阴线**：主力出货
- **板块退潮**：跟风股开始跌停
- **达到预期涨幅**：严格执行止盈

### 3. 仓位管理
- 龙头仓位可适当重
- 分批建仓
- 保留子弹应对变盘

## 龙头股心法

### "买在分歧，卖在一致"
- 一致预期时往往是高点
- 分歧时才是机会

### "强者恒强"
- 龙头股不轻易言顶
- 顺势而为

### "只做龙头，不做杂毛"
- 龙头股有资金持续关注
- 杂毛股容易被抛弃

### "慢就是快"
- 宁可错过，不可做错
- 稳定盈利比暴利更重要

## 风险控制

1. **仓位控制**
   - 单只股票不超过30%
   - 留有补仓资金

2. **止损纪律**
   - 亏损8%必须止损
   - 不抄底，不补仓摊薄成本

3. **情绪管理**
   - 不盲目追高
   - 不YY（意淫）龙头
   - 尊重市场选择

## 常见错误

1. **龙头买不到，买到就是坑**
   - 这是散户常见的误区
   - 实际上龙头股会有分歧机会

2. **幻想从头拿到尾**
   - 没有人能卖在最高点
   - 达到预期就要走

3. **盲目抄底**
   - 龙头股的回调可能是深渊
   - 不要接飞刀

4. **重仓持有一只股**
   - 风险过于集中
   - 一旦失败损失惨重

## 实战案例

### 经典龙头走势
```
第一阶段：连续涨停（加速期）
    ↓
第二阶段：高位震荡（分歧期）
    ↓
第三阶段：高位反包或直接回落
    ↓
第四阶段：龙回头（反弹期）或结束
```

### 情绪周期配合
- 情绪上升期：龙头股涨幅最大
- 情绪高潮期：龙头股见顶
- 情绪退潮期：回避高位龙头
