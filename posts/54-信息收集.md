## 一、信息收集概述

渗透测试的第一步是尽可能多地收集目标信息。信息收集占整个渗透测试工作量的 **40%~60%**，信息越全面，发现漏洞的可能性越大。

### 信息收集分类

```
被动收集（不直接接触目标）
  → Google Dorking、Shodan、DNS 查询、Whois、社交媒体

主动收集（直接与目标交互）
  → 端口扫描、服务识别、目录枚举、漏洞探测
```

## 二、DNS 信息收集

```bash
# 基本 DNS 查询
dig example.com              # A 记录
dig example.com MX          # 邮件记录
dig example.com TXT         # SPF/TXT 记录
dig example.com NS          # 域名服务器
dig -x 8.8.8.8             # PTR 记录（反向查询）
dig @8.8.8.8 example.com    # 指定 DNS 服务器

# 区域传输
dig @ns1.example.com example.com AXFR

# Whois 查询
whois example.com
whois 8.8.8.8

# DNSenum（综合 DNS 枚举）
dnsenum example.com
dnsenum -f /usr/share/dnsenum/dns.txt -dnsserver 8.8.8.8 example.com

# DNSdumpster（在线）
# https://dnsdumpster.com/

# Sublist3r（子域名枚举）
sublist3r -d example.com -o subdomains.txt
```

## 三、Nmap 端口扫描

### 基本扫描

```bash
# ping 扫描（发现存活主机）
nmap -sn 192.168.1.0/24

# 快速扫描常用端口
nmap -F 192.168.1.100

# 全端口扫描（-p- = -p 1-65535）
nmap -p- 192.168.1.100

# 服务版本检测
nmap -sV 192.168.1.100

# 操作系统检测
nmap -O 192.168.1.100

# 全面扫描（脚本 + 版本 + OS + 快速）
nmap -A 192.168.1.100

# 脚本扫描（默认脚本）
nmap -sC 192.168.1.100
```

### 常用 Nmap 脚本

```bash
# Web 相关
nmap --script http-enum 192.168.1.100        # 目录枚举
nmap --script http-title 192.168.1.100       # 获取网站标题
nmap --script http-robots.txt 192.168.1.100  # 获取 robots.txt
nmap --script http-sql-injection 192.168.1.100  # SQL注入检测
nmap --script http-xssed 192.168.1.100      # 检测已知 XSS

# SMB 相关
nmap --script smb-vuln* 192.168.1.100        # SMB 漏洞检测
nmap --script smb-os-discovery 192.168.1.100 # SMB 系统信息

# DNS 相关
nmap --script dns-zone-transfer -p 53 --script-args dns-zone-transfer.server=ns1.example.com -d example.com

# SSL/TLS 检测
nmap --script ssl-enum-ciphers -p 443 example.com
nmap --script ssl-cert,ssl-heartbleed -p 443 example.com
```

### Nmap 输出格式

```bash
# 输出到文件
nmap -oA scan_result 192.168.1.100  # 同时输出 3 种格式
nmap -oN result.nmap 192.168.1.100   # 正常格式
nmap -oX result.xml 192.168.1.100    # XML 格式
nmap -oG result.gnmap 192.168.1.100  # Grepable 格式

# Zenmap（GUI 版本）
zenmap
```

## 六、网络扫描工具对比

| 工具 | 特点 | 适用场景 |
|------|------|---------|
| Nmap | 最全面，脚本丰富 | 端口/服务扫描 |
| Masscan | 超高速，异步 | 大型网络快速扫描 |
| RustScan | 现代，兼容 Nmap | 快速端口扫描 |
| Unicornscan | 异步 | 高速无状态扫描 |

```bash
# Masscan（比 Nmap 快 3 倍）
masscan -p1-10000 192.168.1.0/24 --rate=10000

# RustScan（更现代）
rustscan -b 1000 -t 2000 -p 1-10000 192.168.1.100
```
