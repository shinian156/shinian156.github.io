## 一、爬虫基础与环境搭建

### 1.1 爬虫基本原理

爬虫（Web Crawler）是一种自动获取网页内容的程序，其工作流程：

```
发送请求 → 接收响应 → 解析内容 → 存储数据 → 遍历链接（循环）
```

### 1.2 法律与伦理

```python
# 爬虫基本准则
"""
1. 遵守 robots.txt 协议
2. 控制请求频率，避免对服务器造成负担
3. 不爬取敏感个人信息
4. 尊重版权，合理使用数据
5. 涉及商业用途时获取授权
"""
```

```python
# myspider/spiders/example.py
import scrapy

class ExampleSpider(scrapy.Spider):
    name = "example"
    allowed_domains = ["example.com"]
    start_urls = ["https://example.com"]

    def parse(self, response):
        # 提取当前页面数据
        for item in response.css(".item"):
            yield {
                "title": item.css(".title::text").get(),
                "url": item.css("a::attr(href)").get(),
                "price": item.css(".price::text").get(),
            }

        # 翻页
        next_page = response.css("a.next::attr(href)").get()
        if next_page:
            yield response.follow(next_page, callback=self.parse)
```

## 六、异步爬虫 aiohttp

### 6.1 基本用法

```python
import asyncio
import aiohttp

async def fetch(session, url):
    async with session.get(url) as response:
        return await response.text()

async def main():
    urls = [f"https://example.com/page/{i}" for i in range(1, 11)]
    async with aiohttp.ClientSession() as session:
        tasks = [fetch(session, url) for url in urls]
        htmls = await asyncio.gather(*tasks)
        for html in htmls:
            print(len(html))

asyncio.run(main())
```

### 6.2 异步解析

```python
import asyncio
import aiohttp
from bs4 import BeautifulSoup

async def parse_page(session, url):
    async with session.get(url) as response:
        html = await response.text()
        soup = BeautifulSoup(html, "lxml")
        items = soup.select(".item")
        return [{"title": i.select_one(".title").get_text(strip=True)} for i in items]

async def main():
    urls = [f"https://example.com/{i}" for i in range(1, 100)]
    connector = aiohttp.TCPConnector(limit=100)
    async with aiohttp.ClientSession(connector=connector) as session:
        tasks = [parse_page(session, url) for url in urls]
        results = await asyncio.gather(*tasks)
        for items in results:
            print(items)
```

### 6.3 异步存储

```python
import asyncio
import aiohttp
import aiofiles
import json

async def save_to_file(filename, data):
    async with aiofiles.open(filename, "a", encoding="utf-8") as f:
        await f.write(json.dumps(data, ensure_ascii=False) + "\n")

async def main():
    urls = [f"https://example.com/{i}" for i in range(1, 100)]
    connector = aiohttp.TCPConnector(limit=100)
    semaphore = asyncio.Semaphore(20)  # 限制并发数

    async with aiohttp.ClientSession(connector=connector) as session:
        async def fetch_with_limit(url):
            async with semaphore:
                async with session.get(url) as response:
                    return await response.json()

        tasks = [fetch_with_limit(url) for url in urls]
        results = await asyncio.gather(*tasks)

        for data in results:
            await save_to_file("data.jsonl", data)
```

```python
def get_proxy():
    # 从代理池服务获取
    response = requests.get("http://proxy-pool:5000/get")
    return response.json().get("proxy")

def fetch_with_proxy(url):
    proxy = get_proxy()
```

### 7.3 请求频率控制

```python
import time
import random

class RateLimiter:
    def __init__(self, min_delay=1, max_delay=3):
        self.min_delay = min_delay
        self.max_delay = max_delay

    def wait(self):
        delay = random.uniform(self.min_delay, self.max_delay)
        time.sleep(delay)

limiter = RateLimiter(min_delay=1, max_delay=3)

for url in urls:
    limiter.wait()
    response = requests.get(url)
```

### 8.3 实战案例：豆瓣电影爬虫

```python
import requests
from bs4 import BeautifulSoup
import time

class DoubanSpider:
    def __init__(self):
        self.session = requests.Session()
        self.session.headers.update({
            "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36",
            "Referer": "https://movie.douban.com/"
        })

    def get_movies(self, start=0):
        url = f"https://movie.douban.com/top250?start={start}"
        response = self.session.get(url, timeout=10)
        soup = BeautifulSoup(response.text, "lxml")

        movies = []
        for item in soup.select(".item"):
            movie = {
                "title": item.select_one(".title").get_text(strip=True),
                "rating": item.select_one(".rating_num").get_text(strip=True),
                "quote": item.select_one(".inq").get_text(strip=True) if item.select_one(".inq") else "",
                "url": item.select_one("a").get("href")
            }
            movies.append(movie)
        return movies

    def crawl_all(self):
        all_movies = []
        for start in range(0, 250, 25):
            print(f"正在爬取第 {start//25 + 1} 页...")
            movies = self.get_movies(start)
            all_movies.extend(movies)
            time.sleep(2)  # 避免被封
        return all_movies

spider = DoubanSpider()
movies = spider.crawl_all()
print(f"共爬取 {len(movies)} 部电影")
```

## 九、高频面试题汇总

1. **Scrapy vs Requests + BeautifulSoup 如何选择？**
   - Scrapy：大型项目、需要并发爬取、内置去重/中间件/存储框架
   - Requests + BS4：小规模、简单页面、快速原型开发

2. **如何避免被封IP？**
   - 控制请求频率（设置 DOWNLOAD_DELAY）
   - 使用代理池轮换 IP
   - 设置随机的 User-Agent
   - 尽量在非高峰期爬取

3. **如何处理动态加载页面？**
   - 使用 Selenium/Playwright 等浏览器自动化工具
   - 寻找 API 接口（通常比页面爬取更稳定）
   - 使用 Splash 等 JS 渲染服务

4. **如何实现增量爬取？**
   ```python
   # URL 去重
   seen_urls = set()
   def is_new_url(url):
       if url in seen_urls:
           return False
       seen_urls.add(url)
       return True

   # 增量更新数据库
   def upsert_item(item):
       cursor.execute("""
           INSERT INTO table (title, url) VALUES (%s, %s)
           ON DUPLICATE KEY UPDATE title=%s
       """, (item["title"], item["url"], item["title"]))
   ```

5. **分布式爬虫实现？**
   - 使用 Scrapy-Redis：基于 Redis 实现请求队列共享
   - 使用 Scrapy-Cluster：基于 Kafka/ZooKeeper 的分布式方案
   - 主从模式：Master 分配任务，多个 Worker 执行

6. **反反爬策略？**
   - 代理池轮换 + 随机延时
   - 模拟登录获取 Cookie
   - IP 代理 + UA 轮换
   - 验证码识别（OCR / 打码平台）
   - 签名算法逆向（JS 补环境或直接调用）

7. **爬虫如何遵守 robots.txt？**
   ```python
   from urllib import robotparser

   rp = robotparser.RobotFileParser()
   rp.set_url("https://example.com/robots.txt")
   rp.read()

   if rp.can_fetch("*", url):
       # 可以爬取
       pass
   ```
