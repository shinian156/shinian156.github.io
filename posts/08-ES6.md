## 一、let、const 与 var 的区别

### 1. 变量声明对比

| 特性 | var | let | const |
|------|-----|-----|-------|
| 作用域 | 函数作用域 | 块级作用域 | 块级作用域 |
| 变量提升 | ✅ 是 | ✅ 是（TDZ） | ✅ 是（TDZ） |
| 重复声明 | ✅ 允许 | ❌ 不允许 | ❌ 不允许 |
| 重新赋值 | ✅ 允许 | ✅ 允许 | ❌ 不允许 |
| 全局属性 | ✅ 是 | ❌ 否 | ❌ 否 |

### 2. 作用域示例
```javascript
// var：函数作用域
function fn() {
  if (true) {
    var x = 1
  }
  console.log(x)  // 1（if 块外也能访问）
}

// let/const：块级作用域
function fn() {
  if (true) {
    let x = 1
    const y = 2
  }
  console.log(x)  // ReferenceError
  console.log(y)  // ReferenceError
}
```

### 3. 变量提升与 TDZ（Temporal Dead Zone）
```javascript
// var：提升为 undefined
console.log(x)  // undefined
var x = 1

// let/const：TDZ，提升但不可访问
console.log(y)  // ReferenceError
let y = 1

console.log(z)  // ReferenceError
const z = 1
```

### 4. 重复声明
```javascript
var a = 1
var a = 2  // ✅ 允许

let b = 1
let b = 2  // ❌ SyntaxError

const c = 1
const c = 2  // ❌ SyntaxError
```

### 5. 全局变量
```javascript
var global1 = 'var'
let global2 = 'let'
const global3 = 'const'

console.log(window.global1)  // 'var'
console.log(window.global2)  // undefined
console.log(window.global3)  // undefined
```

## 二、解构赋值

### 1. 数组解构
```javascript
// 基本用法
const [a, b, c] = [1, 2, 3]
console.log(a, b, c)  // 1 2 3

// 忽略某些值
const [first, , third] = [1, 2, 3]
console.log(first, third)  // 1 3

// 默认值
const [x = 0, y = 0] = [1]
console.log(x, y)  // 1 0

// 剩余元素
const [head, ...tail] = [1, 2, 3, 4, 5]
console.log(head, tail)  // 1 [2, 3, 4, 5]

// 交换变量
let m = 1, n = 2
[m, n] = [n, m]
console.log(m, n)  // 2 1
```

### 2. 对象解构
```javascript
// 基本用法
const { name, age } = { name: 'John', age: 30 }
console.log(name, age)  // 'John' 30

// 重命名
const { name: userName, age: userAge } = { name: 'John', age: 30 }
console.log(userName, userAge)  // 'John' 30

// 默认值
const { x = 0, y = 0 } = { x: 1 }
console.log(x, y)  // 1 0

// 剩余属性
const { a, b, ...rest } = { a: 1, b: 2, c: 3, d: 4 }
console.log(rest)  // { c: 3, d: 4 }

// 嵌套解构
const user = {
  info: {
    name: 'John',
    address: {
      city: 'NYC'
    }
  }
}
const { info: { name, address: { city } } } = user
console.log(name, city)  // 'John' 'NYC'
```

### 3. 函数参数解构
```javascript
// 数组参数
function sum([a, b, c]) {
  return a + b + c
}
console.log(sum([1, 2, 3]))  // 6

// 对象参数
function greet({ name, greeting = 'Hello' }) {
  console.log(`${greeting}, ${name}!`)
}
greet({ name: 'John' })  // 'Hello, John!'

// 嵌套解构
function fn({ data: { items } }) {
  console.log(items)
}
fn({ data: { items: [1, 2, 3] } })  // [1, 2, 3]
```

## 三、箭头函数

### 1. 基本语法
```javascript
const add = (a, b) => a + b
const square = x => x * x
const greet = () => console.log('Hello')
const getUser = () => ({ name: 'John', age: 30 })
```

### 2. this 指向
```javascript
// 箭头函数没有自己的 this，继承外层
const obj = {
  name: 'John',
  fn() {
    setTimeout(() => {
      console.log(this.name)  // 'John'（继承 fn 的 this）
    }, 1000)
  }
}

// 箭头函数不能作为构造函数
const Person = (name) => { this.name = name }
const person = new Person('John')  // TypeError

// 箭头函数没有 arguments
const fn = () => {
  console.log(arguments)  // ReferenceError
}
```

### 3. 不能用作对象方法
```javascript
const obj = {
  name: 'John',
  sayName: () => {
    console.log(this.name)  // undefined（this 指向全局）
  }
}
obj.sayName()
```

## 五、扩展运算符与剩余运算符

### 1. 扩展运算符
```javascript
// 数组展开
const arr1 = [1, 2, 3]
const arr2 = [...arr1, 4, 5]
console.log(arr2)  // [1, 2, 3, 4, 5]

// 对象展开
const obj1 = { name: 'John' }
const obj2 = { ...obj1, age: 30 }
console.log(obj2)  // { name: 'John', age: 30 }

// 函数调用展开
function sum(a, b, c) {
  return a + b + c
}
const numbers = [1, 2, 3]
console.log(sum(...numbers))  // 6

// 数组浅拷贝
const arr = [1, 2, 3]
const copy = [...arr]

// 对象浅拷贝
const obj = { name: 'John', age: 30 }
const copy = { ...obj }
```

### 2. 剩余运算符
```javascript
// 数组剩余参数
function sum(...args) {
  return args.reduce((a, b) => a + b, 0)
}
console.log(sum(1, 2, 3, 4, 5))  // 15

// 对象剩余属性
const { a, b, ...rest } = { a: 1, b: 2, c: 3, d: 4 }
console.log(rest)  // { c: 3, d: 4 }

// 数组解构剩余
const [first, ...tail] = [1, 2, 3, 4, 5]
console.log(tail)  // [2, 3, 4, 5]
```

## 七、Symbol

### 1. 基本用法
```javascript
const sym1 = Symbol('id')
const sym2 = Symbol('id')
console.log(sym1 === sym2)  // false（每次创建都是唯一）

const obj = {
  [sym1]: 'value1',
  [sym2]: 'value2'
}
console.log(obj[sym1])  // 'value1'
```

### 2. 内置 Symbol
```javascript
// Symbol.iterator：迭代器
const arr = [1, 2, 3]
const iterator = arr[Symbol.iterator]()
console.log(iterator.next())  // { value: 1, done: false }

// Symbol.toStringTag：自定义 toString
class MyArray {
  get [Symbol.toStringTag]() {
    return 'MyArray'
  }
}
console.log(Object.prototype.toString.call(new MyArray()))  // '[object MyArray]'
```

### 3. 应用场景
```javascript
// 1. 对象属性私有化
const PRIVATE_KEY = Symbol('private')

class MyClass {
  constructor() {
    this[PRIVATE_KEY] = 'secret'
  }

  getPrivate() {
    return this[PRIVATE_KEY]
  }
}

// 2. 消除魔术字符串
const TYPE = {
  SUCCESS: Symbol('success'),
  ERROR: Symbol('error')
}

function log(type, message) {
  if (type === TYPE.SUCCESS) {
    console.log('✅', message)
  }
}
```

## 八、Promise

### 1. 基本用法
```javascript
const promise = new Promise((resolve, reject) => {
  setTimeout(() => {
    resolve('Success')
    // reject('Error')
  }, 1000)
})

promise
  .then(result => console.log(result))
  .catch(error => console.error(error))
  .finally(() => console.log('Done'))
```

### 2. Promise 状态
- **Pending**：初始状态
- **Fulfilled**：操作成功（resolve）
- **Rejected**：操作失败（reject）
- 状态一旦改变，不可逆转

### 3. Promise.all
```javascript
const p1 = Promise.resolve(1)
const p2 = Promise.resolve(2)
const p3 = Promise.resolve(3)

Promise.all([p1, p2, p3])
  .then(results => console.log(results))  // [1, 2, 3]
  .catch(error => console.error(error))

// 一个失败则全部失败
Promise.all([p1, p2, Promise.reject('error')])
  .catch(error => console.log(error))  // 'error'
```

### 4. Promise.race
```javascript
const p1 = new Promise(resolve => setTimeout(() => resolve(1), 1000))
const p2 = new Promise(resolve => setTimeout(() => resolve(2), 500))

Promise.race([p1, p2])
  .then(result => console.log(result))  // 2（p2 先完成）
```

### 5. Promise.allSettled
```javascript
Promise.allSettled([
  Promise.resolve(1),
  Promise.reject('error'),
  Promise.resolve(3)
]).then(results => {
  console.log(results)
  // [
  //   { status: 'fulfilled', value: 1 },
  //   { status: 'rejected', reason: 'error' },
  //   { status: 'fulfilled', value: 3 }
  // ]
})
```

## 九、Async/Await

### 1. 基本用法
```javascript
async function fetchData() {
  try {
    const response = await fetch('/api/data')
    const data = await response.json()
    return data
  } catch (error) {
    console.error(error)
  }
}
```

### 2. 错误处理
```javascript
// try/catch
async function fn() {
  try {
    const data = await fetchData()
  } catch (error) {
    console.error(error)
  }
}

// 捕获多个 await 错误
async function fn() {
  const [data1, data2] = await Promise.allSettled([
    fetch('/api/data1'),
    fetch('/api/data2')
  ])
}
```

### 3. 并行执行
```javascript
// 错误：串行执行
async function sequential() {
  const res1 = await fetch('/api/1')
  const res2 = await fetch('/api/2')
}

// 正确：并行执行
async function parallel() {
  const [res1, res2] = await Promise.all([
    fetch('/api/1'),
    fetch('/api/2')
  ])
}
```

### 4. async 函数返回值
```javascript
async function fn() {
  return 'value'
}

const promise = fn()
console.log(promise instanceof Promise)  // true

fn().then(result => console.log(result))  // 'value'
```

## 十、Class 类

### 1. 基本语法
```javascript
class Person {
  constructor(name, age) {
    this.name = name
    this.age = age
  }

  sayHello() {
    console.log(`Hello, I'm ${this.name}`)
  }

  static getSpecies() {
    return 'Homo sapiens'
  }
}

const person = new Person('John', 30)
person.sayHello()  // 'Hello, I'm John'
console.log(Person.getSpecies())  // 'Homo sapiens'
```

### 2. 继承
```javascript
class Animal {
  constructor(name) {
    this.name = name
  }

  eat() {
    console.log(`${this.name} is eating`)
  }
}

class Dog extends Animal {
  constructor(name, breed) {
    super(name)  // 调用父类构造函数
    this.breed = breed
  }

  bark() {
    console.log('Woof!')
  }

  eat() {
    super.eat()  // 调用父类方法
    console.log('Dog eating style')
  }
}
```

### 3. Getter/Setter
```javascript
class Circle {
  constructor(radius) {
    this._radius = radius
  }

  get radius() {
    return this._radius
  }

  set radius(value) {
    if (value > 0) {
      this._radius = value
    }
  }

  get area() {
    return Math.PI * this._radius ** 2
  }
}

const circle = new Circle(5)
console.log(circle.area)  // 78.53981633974483
circle.radius = 10
```

### 4. 私有字段
```javascript
class Person {
  #name = 'John'  // 私有字段

  getName() {
    return this.#name
  }
}

const person = new Person()
console.log(person.getName())  // 'John'
console.log(person.#name)  // SyntaxError
```

## 十一、Map 与 Set

### 1. Map
```javascript
const map = new Map()

// 添加
map.set('name', 'John')
map.set('age', 30)
map.set('name', 'Jane')  // 覆盖

// 获取
console.log(map.get('name'))  // 'Jane'

// 检查
console.log(map.has('name'))  // true

// 删除
map.delete('age')

// 大小
console.log(map.size)  // 1

// 清空
map.clear()

// 遍历
map.forEach((value, key) => console.log(key, value))
for (const [key, value] of map) {
  console.log(key, value)
}

// 特殊：对象作为 key
const obj = {}
map.set(obj, 'value')
console.log(map.get(obj))  // 'value'
```

### 2. Set
```javascript
const set = new Set()

// 添加
set.add(1)
set.add(2)
set.add(1)  // 重复值不添加

// 检查
console.log(set.has(1))  // true

// 删除
set.delete(2)

// 大小
console.log(set.size)  // 1

// 数组去重
const arr = [1, 2, 2, 3, 3, 3]
const unique = [...new Set(arr)]
console.log(unique)  // [1, 2, 3]

// 交集
const set1 = new Set([1, 2, 3])
const set2 = new Set([2, 3, 4])
const intersection = [...set1].filter(x => set2.has(x))
console.log(intersection)  // [2, 3]

// 并集
const union = [...new Set([...set1, ...set2])]
console.log(union)  // [1, 2, 3, 4]
```

## 十二、迭代器与生成器

### 1. 迭代器
```javascript
const arr = [1, 2, 3]
const iterator = arr[Symbol.iterator]()

console.log(iterator.next())  // { value: 1, done: false }
console.log(iterator.next())  // { value: 2, done: false }
console.log(iterator.next())  // { value: 3, done: false }
console.log(iterator.next())  // { value: undefined, done: true }
```

### 2. 生成器
```javascript
function* generator() {
  yield 1
  yield 2
  yield 3
}

const gen = generator()
console.log(gen.next())  // { value: 1, done: false }
console.log(gen.next())  // { value: 2, done: false }
console.log(gen.next())  // { value: 3, done: false }
console.log(gen.next())  // { value: undefined, done: true }
```

### 3. 生成器应用
```javascript
// 斐波那契数列
function* fibonacci() {
  let [a, b] = [0, 1]
  while (true) {
    yield a
    ;[a, b] = [b, a + b]
  }
}

const fib = fibonacci()
console.log(fib.next().value)  // 0
console.log(fib.next().value)  // 1
console.log(fib.next().value)  // 1
console.log(fib.next().value)  // 2
```

## 十三、ES2020-ES2024 新特性

### 1. 可选链操作符（?.）
```javascript
const user = {
  info: {
    address: {
      city: 'NYC'
    }
  }
}

console.log(user.info?.address?.city)  // 'NYC'
console.log(user.info?.phone?.number)  // undefined（不报错）
```

### 2. 空值合并运算符（??）
```javascript
const value = null ?? 'default'  // 'default'
const value = undefined ?? 'default'  // 'default'
const value = 0 ?? 'default'  // 0
const value = false ?? 'default'  // false
```

### 3. BigInt
```javascript
const big = 9007199254740991n
const sum = big + 1n
console.log(sum)  // 9007199254740992n
```

### 4. String.prototype.replaceAll
```javascript
const str = 'hello world'
const result = str.replaceAll('l', 'x')
console.log(result)  // 'hexxo worxd'
```

### 5. Promise.any
```javascript
Promise.any([
  Promise.reject('error1'),
  Promise.resolve(1),
  Promise.resolve(2)
]).then(result => console.log(result))  // 1
```

### 6. Object.hasOwn
```javascript
const obj = { name: 'John' }
console.log(Object.hasOwn(obj, 'name'))  // true
```

## 十四、常见知识点

### 1. var、let、const 的区别？
- **var**：函数作用域，变量提升，可重复声明
- **let**：块级作用域，TDZ，不可重复声明，可重新赋值
- **const**：块级作用域，TDZ，不可重复声明，不可重新赋值

### 2. 箭头函数与普通函数的区别？
- 箭头函数没有自己的 this，继承外层
- 箭头函数不能作为构造函数
- 箭头函数没有 arguments
- 箭头函数没有 prototype

### 3. Promise 的状态有哪些？
- Pending（进行中）
- Fulfilled（已成功）
- Rejected（已失败）

### 4. async/await 的作用？
- 同步方式编写异步代码
- 更好的错误处理（try/catch）
- 解决 Promise 回调地狱

### 5. Map 与 Object 的区别？
- **Map**：键可以是任意类型，有序，有 size 属性
- **Object**：键只能是字符串/Symbol，无序，需手动计算大小

### 6. Set 的应用场景？
- 数组去重
- 检查元素是否存在
- 求交集、并集、差集

### 7. 解构赋值的默认值何时生效？
- 当解构的值为 `undefined` 时生效

### 8. 扩展运算符和剩余运算符的区别？
- **扩展运算符（...）**：展开数组/对象
- **剩余运算符（...）**：收集剩余元素

### 9. class 的私有字段怎么定义？
使用 `#` 前缀：`#privateField`

### 10. 可选链操作符（?.）的作用？
避免访问 undefined/null 属性时报错
