# 常见K线图形态

K线图是股票技术分析的基础，不同的K线组合形态传达着不同的市场信息。本文介绍常见的K线形态及其应用。

## 单根K线形态

### 阳线（上涨）
| 类型 | 特征 | 含义 |
|------|------|------|
| 大阳线 | 实体较大，下影线短 | 多方强势，买盘强劲 |
| 小阳线 | 实体较小 | 多方力量较弱 |
| 上影阳线 | 上影线长于实体 | 上涨遇阻，可能回落 |
| 下影阳线 | 下影线长于实体 | 下方支撑强，可能反弹 |

### 阴线（下跌）
| 类型 | 特征 | 含义 |
|------|------|------|
| 大阴线 | 实体较大，上影线短 | 空方强势，卖盘强劲 |
| 小阴线 | 实体较小 | 空方力量较弱 |
| 上影阴线 | 上影线长于实体 | 反弹遇阻 |
| 下影阴线 | 下影线长于实体 | 下方有一定支撑 |

### 十字星
- 开盘价与收盘价接近
- 上下影线较长
- 表示市场犹豫，可能变盘

## 常见组合形态

### 1. 吞没形态

**看涨吞没**
```
特征：下跌趋势中出现一根大阳线，完全吞没前一根阴线
信号：反转信号，多头占优
```

**看跌吞没**
```
特征：上涨趋势中出现一根大阴线，完全吞没前一根阳线
信号：反转信号，空头占优
```

### 2. 孕线形态

**看涨孕线**
```
特征：长阴线后出现一根小阳线，完全在阴线实体内
信号：下跌动能减弱，可能止跌
```

### 3. 早晨之星（启明星）

```
特征：
- 第一天：大阴线（跌）
- 第二天：小实体（可阴可阳）
- 第三天：大阳线（涨）

信号：强烈看涨信号
```

### 4. 黄昏之星

```
特征：
- 第一天：大阳线（涨）
- 第二天：小实体
- 第三天：大阴线（跌）

信号：强烈看跌信号
```

### 5. 锤子线与上吊线

**锤子线**（出现在低位）
- 下影线是实体的2-3倍
- 上影线短或无
- 形似锤子
- **信号**：见底信号

**上吊线**（出现在高位）
- 下影线是实体的2-3倍
- 上影线短或无
- 形似上吊
- **信号**：见顶信号

## 缺口理论

### 普通缺口
- 很快被回补
- 意义不大

### 突破缺口
- 伴随放量
- 短期内不回补
- 表明趋势开始

### 持续缺口
- 表明趋势延续
- 测量作用

### 衰竭缺口
- 趋势末期的跳空
- 很快被回补
- 表明趋势即将结束

## 实战技巧

1. **结合位置判断**
   - 同样的形态在高位和低位意义不同
   - 底部形态更可靠

2. **结合成交量**
   - 放量确认有效性
   - 缩量需谨慎

3. **结合均线**
   - 均线支撑/压力
   - 多周期共振

4. **设置止损**
   - 形态失败及时出局
   - 严格控制风险

## 注意事项

- K线形态不是绝对的
- 需要结合其他分析方法
- 控制仓位，分批操作
- 设置止损位
