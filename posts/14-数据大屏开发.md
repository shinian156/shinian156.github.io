## 1. 数据大屏概述

### 1.1 什么是数据大屏？
数据大屏是一种数据可视化展示形式，通过大屏幕、多图表、实时更新的方式，直观展示关键业务指标和数据分析结果。常见于监控中心、指挥中心、数据驾驶舱等场景。

### 1.2 数据大屏的特点
- **全屏展示**：占据整个屏幕空间
- **实时更新**：数据动态刷新
- **多图表组合**：多种图表类型配合使用
- **响应式适配**：适配不同分辨率
- **视觉冲击力强**：注重设计和交互效果

### 1.3 应用场景
- 业务数据监控
- 运营数据分析
- 设备状态监控
- 智慧城市展示
- 工业4.0监控

## 2. 技术选型

### 2.1 图表库选择

#### ECharts
- ✅ 功能强大，图表类型丰富
- ✅ 性能好，支持大数据量
- ✅ 文档完善，社区活跃
- ✅ 支持地图、3D 等高级功能
- ❌ 文件体积较大

#### Highcharts
- ✅ 图表美观
- ✅ 性能优秀
- ❌ 商业收费
- ❌ 中文支持一般

#### D3.js
- ✅ 灵活性极高
- ✅ 自定义能力强
- ❌ 学习曲线陡峭
- ❌ 开发成本高

#### Three.js (3D可视化)
- ✅ 强大的3D能力
- ✅ 视觉效果好
- ❌ 学习难度大
- ❌ 性能开销大

### 2.2 框架选择
- **Vue + ECharts**：主流选择，生态完善
- **React + ECharts**：React 生态首选
- **原生 JS + ECharts**：轻量级项目

### 2.3 其他工具
- **WebSocket**：实时数据推送
- **Axios**：HTTP 请求
- **Moment.js/Day.js**：时间处理
- **Lodash**：工具函数

### 3.2 核心功能模块

#### 图表封装
```javascript
// 封装通用图表组件
<template>
  <div class="chart-container" ref="chartRef"></div>
</template>

<script>
import * as echarts from 'echarts';

export default {
  props: {
    option: Object,
    theme: String
  },
  mounted() {
    this.chart = echarts.init(this.$refs.chartRef, this.theme);
    this.chart.setOption(this.option);
    window.addEventListener('resize', this.handleResize);
  },
  methods: {
    handleResize() {
      this.chart && this.chart.resize();
    }
  },
  beforeDestroy() {
    window.removeEventListener('resize', this.handleResize);
    this.chart && this.chart.dispose();
  }
}
</script>
```

### 4.1 rem 适配
```javascript
// 根据屏幕宽度设置 rem
function setRem() {
  const width = document.documentElement.clientWidth;
  document.documentElement.style.fontSize = width / 1920 * 16 + 'px';
}
setRem();
window.addEventListener('resize', setRem);
```

### 4.2 scale 适配（推荐）
```javascript
// 按比例缩放整个页面
function resize() {
  const width = 1920;
  const height = 1080;
  const actualWidth = document.documentElement.clientWidth;
  const actualHeight = document.documentElement.clientHeight;

  const scaleX = actualWidth / width;
  const scaleY = actualHeight / height;
  const scale = Math.min(scaleX, scaleY);

  document.body.style.transform = `scale(${scale})`;
  document.body.style.transformOrigin = 'center center';
}
resize();
window.addEventListener('resize', resize);
```

### 7.3 数字滚动动画
```javascript
function animateNumber(element, target) {
  let current = 0;
  const duration = 2000;
  const step = target / (duration / 16);

  function update() {
    current += step;
    if (current < target) {
      element.textContent = Math.floor(current);
      requestAnimationFrame(update);
    } else {
      element.textContent = target;
    }
  }
  update();
}
```

## 8. 实战案例

### 8.1 实时数据大屏
```javascript
// 主组件
<template>
  <div class="dashboard">
    <div class="header">{{ title }}</div>
    <div class="left-panel">
      <chart-card v-for="chart in leftCharts" :option="chart" />
    </div>
    <div class="center-panel">
      <main-chart :option="mainOption" />
    </div>
    <div class="right-panel">
      <chart-card v-for="chart in rightCharts" :option="chart" />
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      title: '数据监控大屏',
      leftCharts: [],
      rightCharts: [],
      mainOption: {}
    }
  },
  mounted() {
    this.initWebSocket();
    this.startTimer();
  },
  methods: {
    initWebSocket() {
      const ws = new WebSocket('ws://localhost:8080/data');
      ws.onmessage = (e) => {
        const data = JSON.parse(e.data);
        this.updateCharts(data);
      };
    },
    startTimer() {
      setInterval(() => {
        this.fetchData();
      }, 5000);
    },
    async fetchData() {
      const data = await this.$http.get('/api/dashboard');
      this.updateCharts(data);
    }
  }
}
</script>
```

## 10. 面试高频题

### Q1: 数据大屏和普通图表页面的区别？
- 全屏展示 vs 普通页面
- 实时性要求更高
- 视觉效果要求更高
- 性能要求更高
- 响应式适配更复杂

### Q2: 如何实现屏幕适配？
- rem 适配
- scale 缩放
- 百分比布局
- CSS Grid/Flexbox

### Q3: 如何优化大屏性能？
- 减少数据量
- 关闭动画
- 使用渐进式渲染
- 合理使用定时器
- 及时清理内存

### Q4: 实时数据如何更新？
- WebSocket 实时推送
- 定时轮询
- Server-Sent Events (SSE)

### Q5: 如何设计大屏布局？
- 确定主题和重点
- 分区域规划
- 图表类型选择
- 信息层次划分
